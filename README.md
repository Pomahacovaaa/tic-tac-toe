A tic tac toe game that looks like the ceramics tic tac toe from pinterest.
Hosted on here and also on itch.io - because the horizons website doesn't let me apload the project link if it's hosted on github.
## How was it made?

* html/js/css
* pinterest
* canva

## Features

* intro page
* two player Tic Tac Toe
* custom theme - ceramics style tic tac toe
* restart button
* background music (can be toggled on/off)
* win and draw detection
